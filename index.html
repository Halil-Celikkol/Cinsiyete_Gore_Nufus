Örnek Kodlar

Veri analizi sırasında kullanılan temel Python kodları. 

import pandas as pd

df = pd.read_csv("Cinsiyete Göre Nüfus.csv")

print(df.head())


import matplotlib.pyplot as plt

plt.plot(df["Yıl"], df["Erkek"])
plt.plot(df["Yıl"], df["Kadın"])
plt.title("Türkiye Nüfus Analizi")
plt.show()

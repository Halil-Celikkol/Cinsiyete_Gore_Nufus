Nasıl Hazırlandı?

Bu proje Python'ın güçlü veri kütüphaneleri kullanılarak geliştirilmiştir:

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_excel("nufus.xlsx")
plt.plot(df["Yıl"], df["Kadın"], label="Kadın")
plt.plot(df["Yıl"], df["Erkek"], label="Erkek")
plt.show()
